Software Installation (Ensenso SDK)

Goal: have the SDK + NxView + NxLib ready so you can open the camera and capture depth.

Checklist
  • • Download the official Ensenso SDK
  • • Install (Windows / Linux)
  • • Open NxView and verify it starts
  • • (Optional) Validate Python bindings
Source

Official guide: Software Installation

Next resultado

When you're done, you should be able to open NxView and see your device in the list without errors.

Success indicators
  • • NxView opens without warnings
  • • Camera appears in the list
  • • You can click “Open” and see Depth

Hardware Installation

Goal: stable Ethernet + correct power so NxView detects the camera.

Steps
  1. Connect Ethernet camera ↔ PC / network
  2. Connect power (according to your model)
  3. If it doesn't show up, check IP / Network Wizard
Source

Official guide: Hardware Installation

Quick troubleshooting

Camera doesn't appear
It's almost always network/IP. Try a direct PC↔camera connection.
It gets hot
Check ambient temperature and flash/illumination cycles.

Basic NxView usage

Goal: open the device, validate Depth/3D, and confirm everything is stable before coding.

Flow
  1. Abrir NxView
  2. Find the camera in the list
  3. Double click / Open
  4. Verify Depth (and/or texture)
Source

Official guide: Basic NxView Usage

Checklist before coding

  • ✅Device opens without warnings
  • ✅Depth looks “clean” (no huge artifacts)
  • ✅Stable network (no disconnects)
